Overview

tinyAgent takes a unique approach to AI agents by treating functions as first-class citizens that can be easily converted into powerful agent tools with just a simple @tool decorator. Built with a hierarchical orchestration system inspired by human organizations, tinyAgent dynamically creates specialized agents for specific tasks while maintaining a perfect balance between security and capability.

Key innovations include:

  • ‚úì Decorator-based tool creation with automatic type validation
  • ‚úì Configurable rate limiting to control API usage
  • ‚úì ElderBrain processing for complex multi-stage tasks
  • ‚úì Hierarchical orchestration with specialized agents

üõ†Ô∏è Basic Usage: Create Tools with Decorators

Turn any Python function into an AI tool with a simple decorator:

# Basic Usage: Create Tools with Decorators from core.decorators import tool from core.agent import Agent # 1. Define a tool with the @tool decorator @tool def format_text(text: str) -> str: """Format text to be properly capitalized with punctuation.""" result = text.strip().capitalize() if not result.endswith((".", "!", "?")): result += "." return result # 2. Create an agent with your tools agent = Agent(tools=[format_text, get_weather]) # 3. Run natural language queries result = agent.run("Format this text: hello world") print(result) # Output: Hello world.

üîí Rate Limiting for API Protection

Protect your external API usage with built-in rate limiting:

# Rate Limiting for API Protection from core.decorators import tool import requests # 1. Add rate limiting to your tool decorator @tool(rate_limit=10, description="Search the web") def search_web(query: str) -> str: """Search the web for the given query.""" response = requests.get("https://api.searchapi.com/search", params={"q": query}, headers={"Authorization": "Bearer $SEARCH_API_KEY"}) return response.json()["results"] # 2. The tool automatically enforces the rate limit # 3. Raises RateLimitExceeded if the limit is exceeded
